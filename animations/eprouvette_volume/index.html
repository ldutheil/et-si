Et si on calculait le volume d'une couronne ?

ACTIONS

Méthode du déplacement d'eau :

1. Note le volume d'eau V1.
2. Glisse l'objet dans l'éprouvette.
3. Note le volume total V2.
4. Le volume de l'objet est V2 - V1.

RÉSULTATS

V1 (Volume d'eau initial) 10.0 mL
V2 (Volume total) 10.0 mL
Vobjet(s) 0.0 mL
